Upcoming Features

These features aren't implemented yet but are coming soon. If you want certain features to be prioritized or have an idea for something you'd want added just send us an email at dev@spaceplanner.app and we'll see what we can do.

Furniture Layouts
The ability to define your furniture peices once and use them in multiple different layouts to quickly switch between when thinking of different setups.
Access Control
Allow other users or groups of users to either view or edit your floor plans
Furniture Rotation Handle
When selected, furniture can be rotated by dragging on a rotation handle
Configurable Input Units
Ability to use only a specific unit, for example, inches for lengths and sizes
